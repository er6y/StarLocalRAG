# LocalRAG - 本地知识库问答系统

这是一个基于Python的本地RAG（检索增强生成）系统，允许用户构建自己的知识库并基于该知识库进行问答。

## 功能特点

### 构建知识库
- 支持多选文件上传（PDF、Word、Excel、PowerPoint、Markdown等格式）
- 支持多种词嵌入模型API（Ollama本地模型、DeepSeek、千问、月之暗面、豆包等）
- 自动保存API密钥，便于下次使用
- 动态获取模型列表
- 实时显示知识库构建进度

### RAG问答
- 支持多种大模型API（Ollama本地模型、DeepSeek、千问、月之暗面、豆包等）
- 知识库选择功能
- 提示词预设功能，可保存常用提示词
- 支持流式输出回答
- 支持停止生成
- 支持新建对话

## 安装依赖

```bash
pip install PyQt5 requests
```

如果需要处理不同类型的文档，还需要安装以下依赖：

```bash
pip install pypdf python-docx openpyxl python-pptx markdown
```

## 使用方法

1. 运行程序：

```bash
python localrag_app.py
```

2. 在"构建知识库"页面：
   - 点击"浏览文件"选择要添加到知识库的文件
   - 选择词嵌入模型API提供商并配置API密钥
   - 点击"获取模型列表"获取可用的词嵌入模型
   - 输入知识库名称
   - 点击"构建知识库"开始构建

3. 在"RAG问答"页面：
   - 选择大模型API提供商并配置API密钥
   - 点击"获取模型列表"获取可用的大模型
   - 从下拉列表中选择已构建的知识库
   - 可以选择或输入提示词预设
   - 在输入框中输入问题
   - 点击"发送"按钮或按Ctrl+Enter发送问题
   - 查看AI回答

## 注意事项

- API密钥会保存在用户主目录下的`.localrag`文件夹中
- 知识库文件保存在程序目录下的`knowledge_bases`文件夹中
- 本程序目前使用模拟数据演示RAG功能，实际使用时需要根据不同API提供商的接口进行适配
